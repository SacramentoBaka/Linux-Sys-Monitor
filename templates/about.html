ℹ️ About This Project

The Linux System Monitor Dashboard is a lightweight real-time monitoring tool for tracking CPU, Memory, and Disk usage on Linux servers. It’s designed to showcase system automation, frontend dashboarding, and backend scripting skills — ideal for DevOps demos, RPA integrations, or IT observability.

🧰 Tech Stack

  • Frontend: HTML5, Chart.js, CSS Grid
  • Backend: Flask (Python)
  • Monitoring Script: Bash (cron + systemd)
  • OS: CentOS 9 Linux

🎯 Key Features

  • Interactive real-time charts (CPU, Memory, Disk)
  • Raw metrics table (last 100 logs)
  • systemd-enabled monitoring script
  • Responsive design for mobile/tablet

🙋 My Role

  • Designed system architecture
  • Wrote Python Flask backend & HTML UI
  • Created systemd service for persistent monitoring
  • Integrated metrics from Linux CLI to Web UI

🚀 This project demonstrates full-stack capability across system automation, backend APIs, and frontend dashboards — The project is available on GitHub.
